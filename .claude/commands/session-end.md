# セッション終了

## 手順

### 1. セッションログの保存
このセッションの会話を以下に保存：
```
docs/sessions/YYYY-MM-DD_[概要].md
```

フォーマット：
```markdown
# Session: YYYY-MM-DD [概要]

## ゴール
[このセッションで目指したこと]

## 実施内容
- [やったこと1]
- [やったこと2]

## 成果
- [できたこと]

## 未完了・課題
- [残っていること]

## 次のアクション
- [ ] [次やること1]
- [ ] [次やること2]

## 会話ログ
[全会話をそのまま記録]
```

### 2. CLAUDE.md の更新
「現在の状況」セクションを更新：
```markdown
## 現在の状況
- 最終更新: YYYY-MM-DD HH:MM
- 進行中タスク: [現在のタスク]
- ブロッカー: [あれば]
- 次のアクション: [次やること]
```

### 3. BACKLOG.md の更新
完了したタスクがあれば状態を更新

### 4. 終了レポート
```
## セッション終了レポート

### 今回の成果
- 

### 未完了
- 

### 次回のアクション
- [ ] 

### Minatoへの確認事項
- [あれば]
```

## 注意
- ログは省略せず全会話を記録
